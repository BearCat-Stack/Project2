Finance Networking Outreach Assistant

A semi-autonomous pipeline that researches finance professionals and drafts elite cold emails — powered by Apollo + Google Gemini.

Python 3.10+ Google Gemini 1.5 Flash Apollo API Playwright
View on GitHub

How It Works

1

Accept a LinkedIn URL or Name + Company as input

2

Enrich contact via Apollo (+ optional LinkedIn proxy)

3

Generate fact-grounded background intelligence

4

Draft a high-conversion cold email via Gemini (QC gated ≥ 8/10)

5

Open Gmail & compose the draft — never auto-sends

Tech Stack

Python 3.10+ google-generativeai requests playwright python-dotenv Apollo.io API

Quick Start

git clone https://github.com/BearCat-Stack/Project2.git cd Project2 python -m venv .venv && source .venv/bin/activate pip install -r requirements.txt playwright install chromium cp .env.example .env # fill in your API keys python networking_assistant.py \ --linkedin-url "https://www.linkedin.com/in/example" \ --sender-name "Your Name" \ --sender-school "Baruch College" \ --sender-major "Finance + CIS" \ --sender-role "Incoming IB Summer Analyst"

Output

Each run produces: a verified email address, a 2–3 sentence career background summary, 2–3 key insights, 3 subject line options, a 90–130 word personalized cold email, and a QC score with rationale. Only emails scoring 8/10 or above are accepted.
